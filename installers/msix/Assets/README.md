# MSIX Assets

Place your MSIX visual assets here (referenced by your `Package.appxmanifest`):

- `Square44x44Logo.targetsize-44.png`
- `Square150x150Logo.scale-200.png`
- `StoreLogo.png`
- `appicon.ico` (if used by the Win32 bridge)
- Any additional scale factors per Windows guidelines.

Recommended: 32-bit color PNG with transparency; provide scale factors (100/200/400).
